# Are You A Good Person?

This is a Processing game where the player goes through moral dilemmas and makes choices to see how “good” they are.  
The game has multiple scenarios, bonus rounds, and keeps track of correct answers.

## How to Run

1. Open the project in [Processing 3.5.4](https://processing.org/).
2. Press the Run button (▶️) to start the game.
3. Use the keyboard to navigate:
   - Space to continue or move between screens
   - Number keys to select options

## Features
- Multiple moral scenarios
- Bonus round
- Results screen showing score and percentage “good”

## Controls
- Space: Continue / advance screens
- Number keys: Select options
- ENTER: Restart game
- BACKSPACE: Exit game
